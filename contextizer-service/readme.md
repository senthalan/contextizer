###MyNews Server

MyNews Rest API server

## Required

    maven
    Java 8
    MongoDB

Create /senthalan/logs/mynews directory for logs

Run mongo in port 27111

Run mvn clean install, It will create mynews-server.zip. 
Unzip and run ./mynewserver start inside bin directory.
